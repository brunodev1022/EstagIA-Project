# EstagIA-Project
Sistema Inteligente de Busca de Estágio para Universitários desenvolvido com React, TypeScript, Node.js, Express, Python Flask, MySQL e MongoDB. Plataforma focada em matchmaking entre alunos e empresas por meio de score de compatibilidade, recomendação automática de vagas e gestão de candidaturas.

## Tecnologias
- React.js + TypeScript
- Node.js + Express
- Python + Flask
- MySQL
- MongoDB

## Funcionalidades
- Cadastro de alunos
- Cadastro de empresas
- Busca automática de vagas
- Score de compatibilidade
- Candidaturas online
- Integração com faculdades
